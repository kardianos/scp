# Three-Phase Research Loop

## The Insight
Gravity, EM, and mass are FIELD properties, not particle properties.
Studying the braid alone will never reveal them. We need three interlocking
investigations that feed each other:

## The Loop

```
    ┌─────────────────────────────────────────────┐
    │                                             │
    ▼                                             │
 Phase A: SUBSTRATE                               │
 What IS the field?                               │
 Lattice structure, symmetry,                     │
 coarse-graining, micro-physics                   │
    │                                             │
    │ constrains                                  │
    ▼                                             │
 Phase B: FIELD                                   │
 What does the field DO?                          │
 Propagation, scattering, effective               │
 metric, thermodynamics, phase transitions        │
    │                                             │
    │ determines                                  │
    ▼                                             │
 Phase C: PARTICLE                                │
 What structures FORM?                            │
 Braids, defects, bound states,                   │
 interactions, emergent properties                │
    │                                             │
    │ constrains                                  │
    └─────────────────────────────────────────────┘
```

Each phase informs the next:
- A→B: Substrate constrains which field behaviors are possible
- B→C: Field behavior determines what particles can form
- C→A: Particle properties constrain what the substrate must be
