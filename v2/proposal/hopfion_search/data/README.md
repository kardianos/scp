# Data Directory — Hopfion Search Numerical Results

All data generated by solvers in `src/`. Organized by category.

## profiles/

Radial soliton profiles. All files are text with comment headers.

### Sigma-model profiles (|q| = rho0 = 1)

| File | Generator | Parameters | Columns |
|------|-----------|------------|---------|
| `profile_sigma_e1.dat` | `bin/radial_solver -e 1` | e=1, rho0=1 | r, f, f', baryon_dens, E2_dens, E4_dens |
| `profile_sigma_e2.dat` | `bin/radial_solver -e 2` | e=2, rho0=1 | r, f, f', baryon_dens, E2_dens, E4_dens |

### Rational map profiles (B=1-4, sigma-model)

| File | Generator | Parameters | Columns |

### Finite-lambda profiles (variable rho(r))

Generated by `bin/finite_lambda_solver`. Under-relaxed alternating iteration (f-shooting + rho-BVP Newton). Profiles bootstrapped from sigma-model through decreasing lambda sequence.

| File | Command | Key results |
|------|---------|-------------|
| `profile_finlam_e2_5000.dat` | `-e 2 -save 5000 -o ...` | rho(0)=0.969, E/E_FB=1.195. Deepest density well at e=2. |
| `profile_finlam_e2_20000.dat` | `-e 2 -save 20000 -o ...` | rho(0)=0.993, E/E_FB=1.224 |
| `profile_finlam_e2_100000.dat` | `-e 2 -save 100000 -o ...` | rho(0)=0.9999, near sigma-model |
| `profile_finlam_e1_8000.dat` | `-e 1 -save 8000 -o ...` | rho(0)=0.996, E/E_FB=1.227 |
| `profile_finlam_e1_10000.dat` | `-e 1 -save 10000 -o ...` | rho(0)=0.998, E/E_FB=1.229 |
| `profile_finlam_100000.dat` | `-e 4 -save 100000 -o ...` | e=4 (original default), rho(0)=0.997 |
| `profile_finlam_10000.dat` | `-e 4 -save 10000 -o ...` | e=4, rho(0)=0.884 |

Columns: `r  f(r)  f'(r)  rho(r)`

## scatter/

Soliton-soliton scattering time series from `bin/scatter`. Leapfrog integrator, product ansatz initialization, Lorentz boost.

| File | Command | Description |
|------|---------|-------------|
| `scatter_same_v0.00.dat` | `-v 0.0` | B+B at rest (control), e=1, N=192, L=10 |
| `scatter_same_v0.30.dat` | `-v 0.3` | B+B repulsive, v=0.3c |
| `scatter_same_v0.50.dat` | `-v 0.5` | B+B repulsive, v=0.5c. Bounce at r=1.222. |
| `scatter_anti_v0.30.dat` | `-anti -v 0.3` | B+Bbar annihilation channel, v=0.3c |
| `scatter_anti_v0.50.dat` | `-anti -v 0.5` | B+Bbar, v=0.5c. Inelastic pass-through, 48% mass radiated. |

Columns: `t  E_pot  E_kin  E_tot  Q  z1  z2  sep  E2  E4  EV`

All scatter runs: e=1, sigma-model profile, N=192 or 256, L=10, lambda=5000, dt=0.001.

## maxwell/

Sourced Maxwell fields from `bin/maxwell`.

| File | Description | Columns |
|------|-------------|---------|
| `maxwell_B1.dat` | B=1 radial EM field | r, rho_B, Q_enc, E_field, phi, rho_E2, rho_E4 |
| `maxwell_B2.dat` | B=2 (hedgehog approx, known bug) | same |
| `maxwell_B3.dat` | B=3 (hedgehog approx, known bug) | same |
| `maxwell_B4.dat` | B=4 (hedgehog approx, known bug) | same |
| `maxwell_angular_B2.dat` | B=2 angular dependence | theta, various angular factors |
| `maxwell_angular_B3.dat` | B=3 angular dependence | theta, various angular factors |
| `maxwell_angular_B4.dat` | B=4 angular dependence | theta, various angular factors |

Note: B>1 maxwell files use hedgehog formula without rational map angular corrections. Q=1 for all B (bug). B=1 is correct: Q=1.000, Coulomb field, E_EM = 0.28% of soliton mass.

## archive/

Early exploratory runs, superseded by the organized profiles above. Kept for reproducibility.

- `profile.dat` — early B=1 sigma-model profile (e=4), no header
- `profile_finite_lambda.dat` — early finite-lambda run (e=4, lambda=100), 2-column format (r, f, rho)
- `profile_lam*.dat` — lambda scan at e=4, 2-column format. Many of these collapsed (rho(0) << 1). Superseded by the bootstrapped solver in finite_lambda.c.
- `field_0000.bin`, `field_final.bin` — 3D field snapshots from gradient flow (binary, N=128)
